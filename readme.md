# Introduction to Anterra Modding (Mostly for Jachro)

## File Structure

```
📁<root>
 ├─📁assets
 │  ├─📁sounds
 │  |  └─📄<sound>.ogg
 │  └─📁textures
 │     └─📄<texture>.png
 ├─📁data 
 │  ├─📁animations
 │  │  └─📄<animation>.json
 │  ├─📁areas
 │  │  └─📄<area>.json
 │  ├─📁entities
 │  │  └─📄<entity>.json
 │  ├─📁items
 │  │  └─📄<item>.json
 │  ├─📁loot_tables
 │  │  └─📄<loot_table>.json
 │  └─📁maps
 │     └─📄<map>.json
 ├─📁scripts
 │  └─📄<script>.cs
 └─📄mod.json
```
